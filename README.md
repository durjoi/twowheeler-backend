# 2wheeler's Backend - A Bicycle Shop

This web application is build with React js, Express js & MongoDB


Project Live Link [2wheeleers](https://twowheelers.web.app/).

## API Endpoints

- /bicycles [GET, POST]
- /bicycles/:id [DELETE]
- /orders [POST]
- /orders/:id [GET, DELETE]
- /orders/:userId [GET]
- /users [POST]
- /users/:id [GET]
- /user/admin [POST]
- /reviews [GET, POST]
